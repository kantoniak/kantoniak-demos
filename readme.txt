This plugin inserts a demo box inside a post. Just add [demo] tag or use media button in the editor.

== Description ==
Demo box can contain any content inside. There are two optional buttons: "Show demo" and "Show all demos". These will show only if appropriate links are provided.

== Installation ==
Works out of the box, just activate installed plugin. No preparation needed.
